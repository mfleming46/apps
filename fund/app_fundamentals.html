Fundamentals Checklist

Item

Load: load data from a .json file.
Save: save data to a .json file.
Append Row: add a new row to the table.
Delete Row(s): remove for which 'Select' is checked.
Shift Right or Left: scroll the current date.
Clear: clear all checkboxes.
You can also edit the row names by typing over what's there
